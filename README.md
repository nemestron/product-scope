# ProductScope

A professional Chrome extension for instant website analysis, built with React, TypeScript, and modern tooling.

## Features

- **Tech Stack Detection**: Comprehensive detection of frameworks, libraries, and build tools
- **SEO Analysis**: Title, meta tags, headings, and canonical URLs
- **Accessibility Scoring**: WCAG compliance checks with detailed scoring
- **Performance Metrics**: Page load time, resource size, DOM complexity analysis
- **Mobile Responsiveness**: Viewport, media queries, and responsive image detection
- **Security Analysis**: HTTPS, CSP, mixed content, and form security checks
- **Social Media Integration**: Detection of Facebook, Twitter, Instagram, and other platforms

## Technical Implementation

- **Frontend**: React + TypeScript with custom hooks for state management
- **Styling**: Custom CSS with dark/light theme toggle
- **Build System**: Vite for fast development and optimized builds
- **Chrome APIs**: Tabs, storage, and content script integration
- **Analysis Engine**: Multi-method detection (scripts, CSS, DOM, global variables)

## UI/UX Design

- **Theme Toggle**: Dark/light mode with persistent user preference
- **Responsive Layout**: Optimized for 384px popup width
- **Smooth Animations**: Fade-in effects and loading states
- **Visual Hierarchy**: Clear information architecture with tabbed interface
- **Color Coding**: Intuitive status indicators (green/yellow/red)
- **Professional Styling**: Clean, minimal design inspired by Apple and Palantir

## Installation

1. Clone this repository
2. Run \
pm install\
3. Run \
pm run build\
4. Load the \dist\ folder as an unpacked extension in Chrome
5. Click the ProductScope icon to analyze any website
