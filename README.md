# ProductScope

A Chrome extension for instant website analysis, including tech stack detection, SEO insights, and accessibility scores.

## Features

- One-click analysis of any website
- Tech stack detection (React, Vue, etc.)
- SEO metrics (title, meta tags, headings)
- Accessibility scores (WCAG compliance)
- Clean, minimalistic black-and-white UI

## Installation

1. Clone this repository
2. Run \
pm install\
3. Run \
pm run build\
4. Load the \dist\ folder as an unpacked extension in Chrome

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS
- Chrome Extension APIs
